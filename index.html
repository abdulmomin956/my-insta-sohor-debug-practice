FAQ

What is the difference between Local Storage and Session Storage?
  • Local storage never expire but Session storage clear when tab is closed.
  • Local storage's capacity is 5MB/10MB while sessionStorage's capacity is 5MB.
  • Local storage is used for short long term memory while session storage is used for short term memory.
  • We must use javascript or manual method to delete localStorage while sessionStorage will be deleted automatically when the tab is closed.
  • Both's data doesn't go to server.
